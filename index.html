Basic Prompt Structure

A good prompt typically contains these key elements:

  • Clear instruction: What you want Claude to do
  • Relevant context: Information needed to complete the task
  • Expected format: How you want the response structured

Being Clear and Direct

Learn to write unambiguous prompts that generate precise responses.

Key Principles:

  • Use specific verbs instead of vague instructions
  • Break complex tasks into smaller steps
  • Specify any constraints or requirements
